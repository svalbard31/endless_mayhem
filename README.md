# Endless_mayhem
Space game based on retro flash games. Using art from the game open source game endless sky. 
(https://github.com/endless-sky/endless-sky) 


# Requriments 
Python 3.11.x and pygame (should setup a vitural enviroment using venv). 

# How to
Run main.py from the source directory. 

One ship uses
wad for control and e to shoot
the other uses 
arrow keys for control  and lshift to shoot. 

Destroying the other ship increases the your score. Getting destroyed decreases yours. 
Have fun :)
